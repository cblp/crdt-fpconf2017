Полурешётка

  • Множество $A$ с операцией $(⋄)$ — полугруппа, если эта операция ассоциативна. $$ (x ⋄ y) ⋄ z = x ⋄ (y ⋄ z) \tag{1} $$
  • Полугруппа $(A, (⋄))$ — полурешётка, если эта эта операция коммутативна и идемпотентна. $$ x ⋄ y = y ⋄ x \tag{2} $$ $$ x ⋄ x = x \tag{3} $$

Что всё это значит? ¯\_(ツ)_/¯

Идемпотентность

$ x ⋄ x = x $

Синхронизация не портит данные

Коммутативность

$ x ⋄ y = y ⋄ x $

Можно синхронизировать в любом направлении

Ассоциативность

$ (x ⋄ y) ⋄ z = x ⋄ (y ⋄ z) $

Можно синхронизировать в любом порядке

Commutative RDT

(CmRDT, op-based)

Тип коммутативен, если от перестановки любых двух конкурентных операций результат не изменяется.

$$ \forall x, y. x \parallel y \implies x \circ y = y \circ x $$

$$ x \parallel y \xlongequal{\mathrm{def}} x \nprec y \land y \nprec x $$

Counter


            data Counter = Increment | Decrement

            instance CmRDT Counter where
                type Payload Counter = Integer
                apply op x = case op of
                    Increment -> x + 1
                    Decrement -> x - 1

                type Intent Counter = Counter
                makeOp i _ = Just (pure i)

2P-Set (2-phase set)


            data TwoPSet a = Add a | Remove a

            instance Ord a => CmRDT (TwoPSet a) where
                type Payload (TwoPSet a) = Set a
                apply op s = case op of
                    Add    a  ->  insert a s
                    Remove a  ->  delete a s

                type Intent (TwoPSet a) = TwoPSet a
                makeOp op s = case op of
                    Remove a | a `notMember` s  ->  Nothing
                    _                           ->  Just (pure op)

            instance Eq a => CausalOrd (TwoPSet a) where
                Add a  `affects`  Remove b  =  a == b
                _      `affects`  _         =  False

            cmrdtLaw = property $ \(s :: Payload op) in1 in2 pid1 pid2 ->
                fromMaybe discard $ do
                    getOp1 <- makeOp @op in1 s
                    getOp2 <- makeOp @op in2 s
                    let (op1, op2) = runLamportClock $
                            (,) <$> runProcess pid1 getOp1
                                <*> runProcess pid2 getOp2
                    pure $
                        concurrent op1 op2 ==>
                        (apply op1 . apply op2) s ===
                        (apply op2 . apply op1) s

Полугруппа с доказательством


            class Semigroup a where
                type (x :: a) <> (y :: a) :: a

                (%<>) ::
                    Sing (x :: a) -> Sing (y :: a) -> Sing (x <> y)

                associativity ::
                    Sing (x :: a) -> Sing (y :: a) -> Sing (z :: a) ->
                    ((x <> y) <> z) :~: (x <> (y <> z))

            (<>) :: (SingKind m, Semigroup m) =>
                    Demote m -> Demote m -> Demote m
            x <> y = withSomeSing x $ \sX ->
                         withSomeSing y $ \sY ->
                             fromSing (sX %<> sY)
